rgon10.general.ntp role – Install and Configure NTP on Linux

Note

This role is part of the rgon10.general collection (version 1.0.0).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install rgon10.general.

To use it in a playbook, specify: rgon10.general.ntp.

Entry point main – Install and Configure NTP on Linux

New in rgon10.general 1.0.0

Synopsis

  • Installs and configures NTP on Linux. At this time, only Debian is supported.

Parameters

Parameter

Comments

ntp_clients

list / elements=string

List of CIDR subnets or IP addresses to provide NTP services to.

Default: []

ntp_config

boolean

Controls if the ntp config is managed by ansible.

Choices:

  • false

  • true ← (default)

ntp_config_file

string

Path to the configuration file for the ntp package. The default should be sane for your OS.

Debian: defaults to /etc/ntpsec/ntp.conf on versions >= 12. Defaults to /etc/ntp.conf on version <= 11

Default: "OS Dependent"

ntp_drift_file

string

Path to the drift file for the ntp package. The default should be sane for your OS.

Debian: defaults to /var/lib/ntpsec/ntp.drift on versions >= 12. Defaults to /var/lib/ntp/drift on version <= 11

Default: "OS Dependent"

ntp_enabled

boolean

Control if the ntp service should be running and enabled for auto start.

Choices:

  • false ← (default)

  • true

ntp_package

string

Name of the NTP package to install. The default should be sane for your OS.

Debian: defaults to ntpsec on versions >= 12. Defaults to ntp on versions <= 11.

Default: "OS Dependent"

ntp_region

string

Set the area for the NTP pool. See ntp.org’s documentation - http://support.ntp.org/bin/view/Servers/NTPPoolServers

Default: ""

ntp_servers

list / elements=string

Override the default NTP servers.

Default: ["0<.ntp_region>.pool.ntp.org", "1<.ntp_region>.pool.ntp.org", "2<.ntp_region>.pool.ntp.org", "3<.ntp_region>.pool.ntp.org"]

ntp_tz

string

Configure the timezone for the system

Default: "Etc/UTC"

ntp_tz_package

string

Name of the NTP package to install. The default should be sane for your OS.

Debian: defaults to tzdata.

Default: "OS Dependent"
